Basic concepts

A HypeDyn story consists of three main components: nodes, links, and rules.

Nodes

Nodes are text fragments which are normally displayed one at a time. A story consists of at least one node. Nodes contain links and node rules.

Links

Links are attached to text in a node. Links have link rules associated with them. Associated link rules are evaluated when the link is clicked by the reader.

Rules

Rules contain conditions and actions. When a rule is evaluated, the condition is checked, and if the condition is true, then the associated actions are triggered

The most common use of a rule is to take the reader to a new node
