Node rules

In addition to attaching rules to links, rules can also be attached to nodes. Node rules are accessed via the Edit node rule button or the Edit node rule menu item under the Edit menu in the node editor.

Node rules are similar to link rules, with the exception that Follow link to and Show in popup actions are not available in node rules. Node rules are triggered when the node is entered.
