Rule editor

The rule editor contains a list of conditions, and list of actions.

Conditions include:

  • Node visited/not visited/previous/not previous
  • Link followed/not followed
  • Comparison of facts (see advanced topics)

Actions include:

  • Follow link to
  • Show in popup
  • Update text (see advanced topics)
  • Update fact (see advanced topics)
