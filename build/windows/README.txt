HypeDyn 2.2a2

To install HypeDyn, unzip the hypedyn_2_2a2_windows folder, and double-click on hypedyn_installer_windows.msi. After installation, a shortcut to HypeDyn will appear on the desktop.

To run HypeDyn, double-click on HypeDyn on the desktop.

If you encounter any bugs, please report them at 
https://bugs.launchpad.net/hypedyn

Note: this is an alpha release. Use with caution. Tutorials have not been updated to reflect changes since 2.1.8.
