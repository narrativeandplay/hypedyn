HypeDyn 2.2.0

To install HypeDyn, unzip the hypedyn_2_2_0_windows folder, and double-click on hypedyn_installer_windows.msi. After installation, a shortcut to HypeDyn will appear on the desktop. Note: if you are installing over an earlier version of HypeDyn, please make sure you restart *before* installing the new version, as otherwise some files may not be replaced properly.

To run HypeDyn, double-click on HypeDyn on the desktop.

If you encounter any bugs, please report them at 
https://bugs.launchpad.net/hypedyn

Note: this is a beta release. Use with caution. Tutorials have been updated to reflect changes since 2.1.8, but are not complete.
