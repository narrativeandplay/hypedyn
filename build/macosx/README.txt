HypeDyn 2.2a2
To install HypeDyn, unzip the hypedyn_2_2a2_macos folder, and copy the entire folder into your Applications folder.

To run HypeDyn, double-click on HypeDyn.app.

If you encounter any bugs, please report them at 
https://bugs.launchpad.net/hypedyn

Note: this is an alpha release. Use with caution. Tutorials have not been updated to reflect changes since 2.1.8.
