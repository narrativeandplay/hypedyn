HypeDyn 2.2.0
To install HypeDyn, unzip the hypedyn_2_2_0_macos folder, and copy the entire folder into your Applications folder.

To run HypeDyn, double-click on HypeDyn.app.

If you encounter any bugs, please report them at 
https://bugs.launchpad.net/hypedyn

Note: this is a beta release. Use with caution. Tutorials have been updated to reflect changes since 2.1.8, but are not complete.
